# StarEmpireSim

StarEmpireSim is a 2D space strategy game built with Python. In this game, players build their empire, collect resources, expand to new planets, and manage fleets to become the strongest force in the galaxy. This project helped me learn how strategy games work and how to build larger systems in Python.

## Features

- Planet exploration and expansion
- Resource collection and management
- Fleet building and movement
- Space battles
- Empire growth system
- Basic enemy AI
- Score and progress tracking

## Tools Used

- Python 3
- Pygame

## How to Run

1. Clone this repository:

git clone https://github.com/TheCuriousBuilder/star-empire-sim

2. Go into the project folder:

cd StarEmpireSim

3. Install Pygame:

pip install pygame

4. Run the game:

python main.py

## Controls

- Arrow Keys = Move / Navigate
- Space = Select / Attack
- Enter = Confirm actions
- Esc = Pause
